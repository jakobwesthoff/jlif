# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.1.0] - 2025-08-18

### Added
- Invert-matches functionality for filtering logic

## [1.0.0] - 2025-08-18

### Added
- Initial release of jlif (JSON Line Formatter)
- Stream processing of JSON data from stdin
- Pretty-printing of JSON with syntax highlighting
- Regex-based filtering with case sensitivity options
- Pass-through mode for non-JSON content
- Configurable buffering for multi-line JSON parsing
- Command-line interface with comprehensive options
